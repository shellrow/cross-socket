# nesmap 
Network mapper for diagnosis and discovery

## Features
- Port Scan
    - Service detection
    - OS detection
- Host Scan
- Ping
- Traceroute
- Subdomain scan

## Supported platforms
- Linux
- macOS
- Windows

## Install
```
cargo install nesmap
```

## Privileges
`nesmap` uses a raw socket which require elevated privileges.  Execute with administrator privileges.
